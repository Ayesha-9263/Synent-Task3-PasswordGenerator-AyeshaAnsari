## 📖 Project Description  
The Secure Password Generator is a command-line interface (CLI) based Python application that generates strong and secure passwords based on user input. The user can define the password length and choose whether to include special characters. The program also evaluates the strength of the generated password based on defined security conditions.

---

## 🎯 Objective  
The objective of this project is to implement Python programming concepts such as loops, conditionals, string manipulation, randomization, and user input handling while building a practical security-based tool.

---

## ⚙️ Features  
- Generate secure random passwords  
- User-defined password length input  
- Option to include or exclude special characters  
- Password strength evaluation (Weak, Medium, Strong, Moderate)  
- Input validation for better user experience  
- Interactive command-line interface (CLI)
